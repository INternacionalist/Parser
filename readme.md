# 💼 IT Вакансии — Уфа

Агрегатор IT-вакансий с hh.ru для Уфы. Парсит вакансии, сохраняет в JSON, показывает на HTML-странице с фильтрами.

## Структура проекта

```
├── main.py          # Точка входа — запускает парсер
├── parser.py        # Парсер hh.ru через официальный API
├── index.html       # Веб-страница с фильтрами
├── vacancies.json   # Генерируется после запуска парсера
├── requirements.txt
└── README.md
```

## Установка

```bash
pip install -r requirements.txt
```

## Запуск

**1. Собрать вакансии:**
```bash
python main.py
```

Скрипт спросит, перепарсить ли заново если файл уже есть.

**2. Открыть страницу:**

Открыть `index.html` в браузере. Если браузер блокирует fetch (CORS), запустить локальный сервер:

```bash
python -m http.server 8000
# затем открыть http://localhost:8000
```

## Что умеет страница

- 🔍 Поиск по названию вакансии
- 🏢 Фильтр по компании (выпадающий список)
- 💰 Ползунок диапазона зарплаты
- 💼 Фильтр по опыту работы
- 🔃 Сортировка: по дате, по зарплате
- ↺ Сброс всех фильтров

## Данные парсера

Парсит по 12 IT-запросам: Python, Java, Frontend, Backend, Fullstack, DevOps, Data Science, QA, Android, iOS, 1C, Sysadmin.

Каждая вакансия содержит:
- `title` — название
- `company` — компания
- `salary` — зарплата (from/to/currency/text)
- `url` — ссылка на hh.ru
- `experience` — требуемый опыт
- `employment` — тип занятости
- `published_at` — дата публикации
